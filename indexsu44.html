2.1.1 problem 3

Let S be event that a man in the US is a smoker and C be event man has cancer. From problem conditions:

P(S) = 0.216 P(C|S) = 23P(C|Sc) P(C|Sc) = 1 23P(C|S)

Lets use Bayes’ theorem

P(S|C) = P(S)P(C|S) P(C) = P(S)P(C|S) P(S)P(C|S) + P(Sc)P(C|Sc) = P(S) ⋅ 23P(C|Sc) P(S) ⋅ 23P(C|Sc) + P(Sc)P(C|Sc) = P(S) 23P(S) + P(Sc) = 23 ⋅ 0.216 23 ⋅ 0.216 + 0.784 ≈ 0.864
