2.3.1 problem 41

Let Gi be the event that the i-th door contains a goat, and let Di be the event that Monty opens door i.

P(G1|D2,G2) = P(G1)P(D2,G2|G1) P(D2,G2) .

P(G1)P(D2,G2|G1) = 2 3 (P(G2|G1)P(D2|G1,G2)) = 2 3 (1 2(p + (1 − p)1 2)) = 2 3 (1 2p + 1 4 − 1 4p) = 1 6(p + 1).

Thus,

P(G1|D2,G2) = 1 6(p + 1) 1 6(p + 1) + 1 3 × 1 2 = p + 1 p + 2.

Note that when p = 1, the result matches that of the basic Monty Hall problem.
