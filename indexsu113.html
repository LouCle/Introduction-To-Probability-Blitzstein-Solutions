3.2.5 problem 20

(a)
P(X = k) =( n k) pk(1 − p)n−k for 0 ≤ k ≤ 3.
(b)
To use the complement of the desired deven, P(X > 0) = 1−P(X = 0) = 1−(1−p)3 = 1−(−p3+3p2−3p+1) = p3−3p2+3p.

To prove the same by Inclusion-Exclusion,

P(X > 0) = ∑ i=13P(I Xi = 1) − 3p2 + P(∩ i=13I Xi = 1) = 3p − 3p2 + p3.

(c)
Since p2 and p3 go to 0 asymptotically faster than p, when p is small, 3p − 3p2 + p3 ≈ 3p.
