3.2.4 problem 19

The pmf of the number of games ending in a draw is P(X = k) =( n k) (0.6)k(0.4)n−k for 0 ≤ k ≤ n.

Let X be the number of games that end in draws. The number of players whose games end in draws is Y = 2X.
