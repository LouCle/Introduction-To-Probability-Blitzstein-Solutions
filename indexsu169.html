4.3.13 problem 53

Let Ij be the indicator variable for tosses j and j + 1 landing heads for 1 ≤ j ≤ 3. Then, the expected number of such pairs is E(X) = ∑ ⁡ j=13P(I j = 1) = 3p2. Var(X) = E(X2) − 9p4.

E(X2) = E((∑ ⁡ j=13I j)2) = E((I 1+I2)2+2(I 1+I2)I3+I32) = E(I 12+2I 1I2+I22+2I 1I3+2I2I3+I32). Note that Ij2 = I j. Thus, E(X2) = (p2 + 2p3 + p2 + 2p3 + 2p3 + p2) = 6p3 + 3p2.

Thus,

Var(X) = 6p3 + 3p2 − 9p4
