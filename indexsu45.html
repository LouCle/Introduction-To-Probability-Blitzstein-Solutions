2.1.2 problem 4

(a)
P(K|R) = P(K)P(R|K) P(R) = P(K)P(R|K) P(K)P(R|K) + P(Kc)P(R|Kc) = p p + (1 − p)1 n
(b)
Since p + (1 − p)1 n ≤ 1, P(K|R) ≥ p with strict equality only when p = 1. This result makes sense, since if Fred gets the answer right, it is more likely that he knew the answer.
