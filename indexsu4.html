1.1.4 problem 4

(a)
There are (n 2) matches.

For a given match, there are two outcomes. Each match has two possible outcomes. We can use the multiplication rule to count the total possible outcomes.

2(n 2)

(b)
Since every player plays every other player exactly once, the number of games is the number of ways to pair up n people.

( n 2)
