4.3.8 problem 46

Let X ∼Geom( 1 13) be the number of non-aces before the first ace. Then, E(X) = 13.

Let Y ∼NHGeom(4, 48, 2) be the number of non-aces before the second ace is drawn. Then, E(X) = rb w+1 = 2∗48 5 = 19.2

Let Z = Y − X. Notice that Z represents the number of non-aces between the first and the second ace. E(Z) = E(Y ) −E(X) = 19.2 − 13 = 6.2.
