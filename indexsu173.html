4.3.17 problem 60

(a)
By the story of the problem, X ∼NHGeom(n,N − n,m). Then, Y = X + m.
(b)
According to part a, E(Y ) = E(X) + m = m(N−n) n+1 + m. The implied indicator variables are the same as in the proof of the expectation of Negative Hypergeometric random variables.
(c)
The problem can be modeled with a Hypergeometric random variable Z ∼HGeom(n,N − n,E(Y )). Then, E(Z) = E(Y ) n N = (m(N−n) n+1 + m) n N = m × N+1 n+1 × n N. Since N+1 n+1 × n N < 1⇒(n + 1)N(n − N) < 0⇒ N−n (n+1)N > 0⇒n < N for positive n and N, E(Z) < m.
